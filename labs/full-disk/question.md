# The Full Filesystem

## Scenario
A critical service on this system has failed to start or is operating extremely slowly. During your investigation, you suspect that the system might be out of disk space.

## Objectives
1. Identify the root cause of the storage exhaustion.
2. Locate the massive rogue file that is filling up the `/var/log` directory.
3. Delete the file to free up space and restore the system to a healthy state.
